publish

var publish = function(topic,
data,
publishReference)

Publish an event.

Parameters

(String) topicThe topic name
(Object) dataThe data you want to publish
(Boolean) publishReferenceIf the data should be passed as a reference to the other modules this parameter has to be set to true.  By default the data object gets copied so that other modules can’t influence the original object.
